# 📖 About

Çalış360 is a Flutter application developed to simplify academic planning for university students.

Instead of using multiple applications for planning, studying and organizing academic tasks, Çalış360 provides a single platform where students can manage their goals, schedules and learning activities.

The project was designed with scalability, maintainability and clean code principles in mind while providing a modern Material Design user experience.

# 🏗 Architecture

The project follows a layered architecture to keep business logic separated from presentation and data access.

```text
UI (Screens & Widgets)
        │
        ▼
Repositories
        │
        ▼
Firebase Services
        │
        ▼
Cloud Firestore
```

The source code is organized into reusable modules, making future development and maintenance easier.
